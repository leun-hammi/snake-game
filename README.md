🐍 Snake Game (Python – Turtle Graphics)


📌 About This Project...
This is a simple Snake Game I built using Python’s Turtle module while learning game development basics. It’s basically my version of the classic snake game where you move around, eat food, grow your snake, and try not to hit the walls.
I built this to practice logic building, loops, and how real-time movement works in Python.

🎮 What it does...
Snake moves using arrow keys
Food spawns randomly on the screen
Every time you eat food, your score increases
Snake grows longer as you score
Game ends if you hit the border
Uses custom images for snake head, body, and background

🛠️ Tech used...
Python
Turtle module
Random module

🧠 What I learned from this...
How game loops actually work
Keyboard event handling in Python
Working with lists for snake body growth
Basic collision detection
Updating game screen in real-time
Managing multiple objects in Turtle
